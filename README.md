# EVOLIZE AI - Sitio Web Oficial

Repositorio oficial del sitio web de presentación de **Evolize AI**, una empresa especializada en diseñar e implementar sistemas inteligentes que transforman la forma en que operan los negocios.

## 🚀 Sobre Evolize AI

Evolize AI no es una agencia de chatbots ni una consultora tecnológica tradicional. Somos arquitectos de sistemas. Analizamos dónde se pierde dinero, tiempo y oportunidades, y diseñamos una arquitectura que:

- Reduce costos ocultos
- Aumenta conversión
- Mejora control
- Genera claridad operativa

### Nuestro Modelo:
**Diagnóstico → Arquitectura → Implementación → Optimización**

## 💻 Sobre este Proyecto (Aspectos Técnicos)

Este proyecto es una landing page estática construida con un enfoque en el rendimiento, la accesibilidad y un diseño premium (estilo _glassmorphism_ y modo oscuro).

### Tecnologías Utilizadas

* **HTML5:** Estructura semántica.
* **CSS3 (Vainilla):** Estilizado completo, incluyendo variables CSS para fácil personalización de temas, Grid/Flexbox para diseño responsivo, y animaciones sutiles.
* **JavaScript (Vainilla):** Interactividad ligera, animaciones al hacer scroll (Intersection Observer) y manipulación del DOM sin dependencias pesadas.
* **Lucide Icons:** Biblioteca de iconos ligeros y consistentes.
* **Google Fonts:** Tipografía primaria "Inter".

### Características del Diseño

* **Dark Mode Nativo:** Esquema de colores oscuros profundos (fondos `#050508`) con gradientes vibrantes para destacar información clave.
* **Glassmorphism:** Uso intensivo de `backdrop-filter: blur`, fondos semi-transparentes y bordes sutiles para emular la textura del cristal.
* **Efectos Visuales:** Sistema de "blobs" (manchas de color desenfocadas) en el fondo animadas mediante keyframes, superposición de cuadrícula de puntos y gradientes de texto.
* **Totalmente Responsivo:** Adaptado a cualquier dispositivo móvil o tablet con media queries optimizadas.

## 🛠️ Instalación y Uso

1. Clona este repositorio:
   ```bash
   git clone https://github.com/tu-usuario/evolize-website.git
   ```
2. Abre el archivo `index.html` en tu navegador web preferido.

*(No requiere Node.js, npm, ni ningún proceso de compilación).*
